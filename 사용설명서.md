# 죽음의 밸런스 토론 배틀 — 사회자 사용설명서

## 📲 접속 주소
- **사회자**: `배포URL/mc` (비밀번호 입력)
- **참가자**: `배포URL/` ← 사회자 화면 QR로 접속

## 🟢 시작 전 (행사 직전)
1. 사회자 화면(`/mc`)을 **대형 화면에 미러링**
2. 화면의 **QR을 참가자에게 보여주기**
3. 참가자: 스캔 → `29기 박상화` 형식 이름 입력 → 입장
4. 우측 상단 **접속 인원** 다 차면 시작

## 🔘 라운드 진행 (버튼 순서대로만 누르면 됨)
화면 아래 **눌러야 할 버튼 1개**만 크게 뜹니다.

1. **▶ 게임 시작** → 첫 질문, 1차 투표 시작
2. *(참가자 선택 대기 — 실시간 카운트 표시)*
3. **🔒 1차 투표 마감**
4. **🎲 대표 랜덤 뽑기** (지원자 있으면 그 사람으로, 없을 때만 클릭)
5. *(대표 2명 각 2~3분 발표)*
6. **🔥 최종 투표 시작** → 재투표
7. **📊 결과 보기** → 1차 vs 최종 비교 + 변화량
8. **➡ 다음 질문으로** (마지막이면 🏁 게임 종료)

→ 질문 5개 반복.

## 🔄 전체 초기화
우측 상단 **[전체 초기화]** → 확인 → 참가자·투표 전부 삭제 후 대기실로.
**리허설 끝나고 본행사 시작 전에 1번** 눌러 테스트 인원 정리.

## ⚠️ 주의
- **행사 중 서버 재시작·재배포 금지** (모든 데이터 초기화됨)
- 늦게 온 사람 → 다음 라운드부터 참여
- 참가자 폰 새로고침해도 자동 복구됨
